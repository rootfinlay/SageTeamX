About TeamX

Admin

As an admin you can create groups and add users. Additional to adding users, you can also change the users password. If you head into the Master files, Domains, you can add domain names, and add descriptions to them. Under Job role groups you can add and delete the groups using the 'Add job role group' and delete them by selecting 'Delete selected job groups' from the action drop down menu. Similiar to the Job role groups, under job roles you can add and remove job roles with the same method used above. Under the header leave type, as an admin you can create leave status and add a description to it. As an admin you can add a leave calender. Under this header you can also see all the notes the users have left, as well as remove them. Underneath the table there is a filter. By selecting a user's name it will filter the table. The next action you can make as admin is adding and removing status's to the Scrum teams. Following scrum team status is Scrum team type, there are three main types arleady listed, and can be removed or added to the table by the admin. As an admin you can also add a scrum team, this will allow you to see the team status, team type, current focus and domain. Following the Scrum team header, there is a Skill header, as an admin you can add remove skills.

User

As a user, you can view info about who's currently on leave within your scrum team and also book holidays for yourself. You cannot currently edit your own information for security reasons but you can message your local Administrator and he can change it for you. All of your data is secure and only the local Administrators have acess to your information. In the next update, we plan to add functionality changes to allow you to book holidays yourself and additional graphical changes to enhance the user experience.
